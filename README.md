# csharp_cic

Disciplina C#

DESCRIÇÃO DO TRABALHO:

Sistema de venda de moto:

O objetivo do trabalho é criar um sistema que cadastrar motos para venda, cadastra vendedores que irão efetuar a venda de uma moto, cadastrar o cliente que irá comprar a moto. Efetuar a venda, contendo um cliente, uma moto e o vendedor que efetuou a venda.
